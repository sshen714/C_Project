# 113-2 計算機程式設計：暗棋小遊戲 (macOS GLUT 版)

這是一個使用 C 語言編寫的暗棋初始化與翻牌邏輯專案。為了在 macOS 環境下順利執行圖形介面，本專案採用了系統內建的 **OpenGL/GLUT** 框架實作。

## 專案功能
1. **棋盤初始化**：自動生成 4x8 棋盤，並將 32 顆棋子（紅黑雙方）隨機打亂放入。
2. **視覺化介面**：使用圖形視窗顯示棋盤，未翻開時顯示為棕色背板，翻開後顯示棋子代號。
3. **玩家操作**：支援滑鼠點擊未翻開的棋子進行翻牌。
4. **電腦對手**：玩家翻牌後，電腦會自動在 0.5 秒延遲後（模擬思考）翻開另一顆棋子。
5. **顏色區分**：紅方棋子（大寫 K, G, E...）以 **紅色** 顯示；黑方棋子（小寫 k, g, e...）以 **白色** 顯示。

## 檔案結構
- `project.c`: 遊戲核心程式碼（包含 OpenGL 繪圖邏輯與遊戲邏輯）。
- `README.md`: 專案說明文件。

## 編譯與執行環境
- **作業系統**: macOS
- **編譯器**: GCC (Apple Clang)
- **依賴框架**: GLUT, OpenGL

### 編譯指令
在終端機輸入以下指令進行編譯：
```bash
gcc project.c -o project -framework GLUT -framework OpenGL
